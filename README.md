# TrustHire Frontend

Frontend minimalista para o TrustHire, integrado com o backend FastAPI e Stripe Checkout.

## Estrutura

- `index.html` → página principal com botão de assinatura  
- `checkout.js` → lógica para criar sessão de checkout Stripe  
- `style.css` → estilos básicos
